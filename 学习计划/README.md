# 专项学习计划总览

这个目录把仓库中的核心技术栈拆成 15 个专项学习计划。每个专项都配有：

- 一份中文学习计划：负责说明学习目标、章节映射、阶段路径、练习方式
- 一个项目内 skill：负责在提问时进入该领域的讲解、陪练、调试、审查、测验模式

## 使用方式

1. 先打开对应专项学习计划，确认当前目标、前置知识、主章节与关联章节
2. 再使用对应 slash skill，围绕当前知识点提问、写代码、排查问题、做测验
3. 学完一个阶段后，把结果同步到 `学习进度.md`
4. 卡住时先回到 `00-学习方法与评估标准.md`，检查是否缺少最小示例、执行流理解或失败路径分析
